## Chemistry in JEE: Strategy

Chemistry is the highest-scoring subject for most JEE toppers. With the right AI tools, it becomes manageable.

**Three branches — three different approaches:**

| Branch | AI use | Priority |
|--------|--------|---------|
| Physical Chemistry | Numericals, derivations | High |
| Organic Chemistry | Mechanism maps, reaction chains | Very High |
| Inorganic Chemistry | Mnemonics, pattern recognition | High |

---

## Branch-Specific AI Prompts

### Physical Chemistry
```
"Give me 10 numerical problems on [TOPIC — e.g. Mole Concept].
 Vary the difficulty. Include problems where unit conversion
 is a common trap. Provide solutions with all steps shown."
```

### Organic Chemistry
```
"Map all the reactions starting from [compound/functional group].
 Show: reagent → product → mechanism type → JEE relevance.
 Format as a visual reaction tree I can trace."
```

### Inorganic Chemistry
```
"Create memory tricks for [group/chapter — e.g. p-Block Group 15].
 I need to remember: oxidation states, colour of compounds,
 key reactions, anomalous behaviour. Make them stick."
```

## Chapter Sequence

```
Physical (do first):
  Mole Concept → Atomic Structure → Chemical Bonding
  → Gaseous State → Thermodynamics → Chemical Equilibrium
  → Ionic Equilibrium → Electrochemistry → Chemical Kinetics

Organic (parallel with Physical):
  Organic Basics + GOC → Alkanes/Alkenes/Alkynes
  → Aromatic Compounds → Haloalkanes → Alcohols/Phenols
  → Aldehydes/Ketones → Carboxylic Acids → Amines → Biomolecules

Inorganic (last, closer to exam):
  Periodic Table + Trends → Chemical Bonding (revisit)
  → s-Block → p-Block (Groups 13–18) → d & f Block
  → Coordination Compounds → Metallurgy
```
