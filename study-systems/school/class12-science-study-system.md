## Class 12 Science — Unique Challenge

You're preparing for:
1. CBSE Board Exams (March)
2. JEE/NEET (April–May) simultaneously

This requires a dual-track strategy.

## Key Insight: Boards + Competitive Together

Don't study them separately. JEE/NEET preparation covers 80% of board syllabus at a deeper level.

**Strategy:** Prepare for JEE/NEET first, then revise CBSE-specific answer formats in the last 6 weeks.

---

## CBSE-Specific AI Prompts

**Answer format training:**
```
"The CBSE Class 12 [Subject] board question is:
 [QUESTION — X marks]
 Write a model answer in CBSE format.
 Bold the keywords. Show diagram placement if needed."
```

**Derivation preparation:**
```
"List all derivations CBSE Class 12 Physics has asked
 in board exams from [UNIT]. For each derivation:
 marks weightage, key steps, diagram if needed."
```

---

## Practical Exam Preparation

```
"I have my Class 12 [subject] practical exam.
 Help me prepare by listing:
 1. All experiments in the syllabus
 2. Common viva questions for each
 3. Key observations and readings I must know
 4. Common errors to avoid"
```
