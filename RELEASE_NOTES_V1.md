# AI Atlas V1 Release Notes

AI Atlas V1 turns the project from a resource collection into a polished static educational platform for JEE, NEET and school students.

## Highlights

- Premium dark AI Atlas interface across generated public pages.
- Site-wide navigation and footer.
- Search across study systems, prompts, workflows, tools, resources, and career guides.
- Page heroes, breadcrumbs, related resources, previous and next navigation.
- Card-based content sections for improved readability.
- Reading progress, table of contents, anchor links, and copy buttons.
- SEO metadata, favicon support, robots file, and search index.

## For Students

- Start from the homepage or search page.
- Jump directly into JEE, NEET, school, prompt, workflow, tools, resources, and career guide sections.
- Use copy buttons on prompt/code blocks for faster studying.

## For Contributors

- Generated pages now share reusable assets in `docs/assets/`.
- Existing HTML can be upgraded with `tools/upgrade-existing-html.js`.
- Search index can be rebuilt with `tools/build-search-index.js`.
