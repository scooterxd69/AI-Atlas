## The Opportunity

AI is creating new roles faster than universities can train people. Students who start building AI skills today will have a significant advantage in 3–5 years.

This guide covers: what roles exist, what skills they actually require, and how to start building now.

---

## Entry-Level AI Roles (2024–2025)

| Role | Skills needed | Typical salary (India) | Typical first employer |
|------|--------------|----------------------|----------------------|
| ML Engineer | Python, mathematics, ML frameworks | ₹8–15 LPA | Startups, IT companies |
| Data Scientist | Statistics, Python, SQL | ₹6–12 LPA | Any data-heavy company |
| AI Product Manager | Domain knowledge, prompt engineering | ₹10–20 LPA | Tech companies |
| Prompt Engineer | Analytical writing, domain expertise | ₹4–10 LPA | AI companies, agencies |
| AI Researcher | Mathematics, Python, research writing | ₹15–30 LPA | Labs, large tech |

## Skill Path for JEE/Engineering Students

```
Foundation (Year 1-2 of college):
  Python basics → NumPy, Pandas → Data visualisation

Intermediate (Year 2-3):
  Machine learning (scikit-learn) → Deep learning basics (PyTorch)
  → Statistics and probability

Advanced (Year 3-4):
  Specialisation (NLP / Computer Vision / RL)
  → Research papers → Open source contributions
  → Portfolio projects with real data
```

## Skill Path for NEET/Biology Students

The intersection of biology and AI is **the biggest opportunity** right now.

```
Foundation:
  Python basics → Bioinformatics tools → R for statistics

Intermediate:
  Computational biology → Protein structure (AlphaFold)
  → Medical imaging AI

Advanced:
  Drug discovery AI → Genomics → Research publications
```
