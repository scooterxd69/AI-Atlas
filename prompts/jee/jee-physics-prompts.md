## Mechanics Prompts

**P-PHY-01 — Rotational Motion Concepts**
```
"Explain moment of inertia from first principles.
 Why is the formula different for different shapes?
 Give me a comparison table of MOI for all standard shapes.
 Then explain the parallel and perpendicular axis theorems
 with derivations."
```

**P-PHY-02 — SHM Problem Solving**
```
"I'm solving a SHM problem: [paste problem].
 Before solving: identify the type of SHM (linear/angular),
 find the equilibrium position, write the restoring force.
 Then solve step by step."
```

**P-PHY-03 — Energy Methods**
```
"Solve [PROBLEM] using energy conservation.
 Identify all energy types present.
 Show the energy transformation at each stage.
 Verify using force method as a check."
```

## Electromagnetism Prompts

**P-PHY-04 — Electric Field Mapping**
```
"For [charge configuration], describe:
 1. Electric field direction and magnitude at key points
 2. Equipotential surfaces
 3. Field lines diagram (describe it verbally)
 4. Analogous gravitational field comparison"
```

**P-PHY-05 — Circuit Analysis**
```
"Analyse this circuit: [describe circuit].
 Use Kirchhoff's laws. Show all equations.
 Identify equivalent resistance.
 Find current through each branch."
```

