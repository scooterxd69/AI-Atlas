## NEET Physics Prompts

**NPC-01 — NCERT Physics Questions**
```
"Create 10 MCQs from NCERT Physics [Chapter] Class [11/12]
 at NEET difficulty (not JEE level).
 Focus on conceptual understanding and formula application.
 Include answer key with one-line explanations."
```

**NPC-02 — Formula Application**
```
"Explain when to use [FORMULA] vs [SIMILAR FORMULA].
 NEET questions often make students confuse these.
 Give 3 examples where the wrong choice leads to wrong answer."
```

## NEET Chemistry Prompts

**NPC-03 — Named Reactions**
```
"List all named reactions in NCERT Chemistry Class 12
 that NEET tests. For each:
 reactants → products, conditions, the name, why it's named so."
```

**NPC-04 — Inorganic Facts**
```
"For NEET Chemistry [Chapter/Group], list all:
 - Formulas of important compounds
 - Colors of compounds
 - Oxidation states
 - Key reactions with equations
 Format as a quick-revision table."
```
