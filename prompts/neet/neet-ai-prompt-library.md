## NCERT Mastery Prompts (N-01 to N-10)

**N-01 — Reading Guide**
```
"Before I read NCERT Biology [Chapter Name] Class [11/12],
 create a reading guide with:
 - 15 key concepts to look for
 - 5 questions to answer while reading
 - Terms I must define precisely
 - Which diagrams to focus on"
```

**N-02 — NCERT Comprehension Check**
```
"Quiz me on NCERT Biology [Chapter] Class [11/12].
 Ask 20 direct questions from the NCERT text.
 After I answer, tell me if my answer matches NCERT exactly.
 Flag any additions or omissions."
```

**N-03 — Keyword Extraction**
```
"Extract all bold and important terms from NCERT Biology
 [Chapter] Class [11/12]. For each term:
 - NCERT definition
 - Why it's important for NEET
 - Has it been asked in NEET PYQs? (if you know)"
```

**N-04 — Exception and Special Cases**
```
"List all exceptions, anomalies and special cases in
 NCERT Biology [Chapter]. NEET loves to test these.
 Format: Exception → Rule it breaks → Why it's different"
```

**N-05 — One-Line Summary Cards**
```
"Summarise every paragraph of NCERT Biology [Chapter]
 in exactly one sentence. These will be my rapid revision cards."
```

**N-06 — Comparison Tables**
```
"From NCERT Biology [Chapter], create all possible
 comparison tables (e.g., Mitosis vs Meiosis).
 These are high-yield for NEET MCQs."
```

**N-07 — Fill in the Blanks**
```
"Convert the key paragraphs of NCERT Biology [Chapter]
 into fill-in-the-blank exercises.
 The blanks should be important terms, numbers, or processes.
 Include answer key."
```

**N-08 — Process Narration**
```
"Narrate [biological process] as a story with a beginning,
 middle and end. Include all steps NCERT mentions.
 Make it vivid enough that I can visualise it happening."
```

**N-09 — NCERT vs Exam Format**
```
"Take the key content from NCERT Biology [Chapter] and
 convert it into NEET MCQ format.
 10 questions, 4 options each, answer key with explanation."
```

**N-10 — Revision in 5 Minutes**
```
"Give me a 5-minute revision of NCERT Biology [Chapter].
 Most important 10 facts only. Bullet points.
 Include any numbers or statistics NCERT mentions."
```

## Diagram Mastery Prompts (D-01 to D-10)

**D-01 — Diagram Description**
```
"Describe the diagram of [biological structure/process]
 for NEET in complete detail.
 Tell me: what it looks like, all labelled parts,
 the function of each part, what questions NEET asks about it."
```

**D-02 — Label Drill**
```
"I've drawn [diagram name]. Give me a label quiz:
 Point to 10 parts by describing their position.
 I'll name the part. You correct me."
```

**D-03 — Process Diagrams**
```
"Walk me through the diagram of [process — e.g. mitosis stages]
 stage by stage. For each stage:
 what is happening, what it looks like,
 which NEET questions ask about this specific stage."
```

## Assertion-Reason Prompts (A-01 to A-10)

**A-01 — Generate A-R Questions**
```
"Generate 15 assertion-reason questions on [Chapter] for NEET.
 Use the standard NEET format:
 (a) Both A and R true, R explains A
 (b) Both true, R doesn't explain A
 (c) A true, R false
 (d) A false, R true
 Mix all four types. Include detailed explanations."
```

**A-02 — Difficult A-R**
```
"Generate 5 tricky assertion-reason questions on [Chapter]
 where the assertion is true but the reason is subtly wrong.
 Explain exactly where students get confused."
```

## PYQ Analysis Prompts (P-01 to P-10)

**P-01 — Chapter PYQ Summary**
```
"Based on NEET PYQs, what are the top 10 concepts from
 [Chapter] that are tested most frequently?
 For each: number of times asked, typical question format,
 what the question tests."
```

**P-02 — PYQ Difficulty Analysis**
```
"Analyse NEET questions from [Chapter] by difficulty.
 Which concepts appear in difficult questions that
 most students get wrong? What's the conceptual gap?"
```

## Mock Analysis Prompts (M-01 to M-10)

**M-01 — Biology Error Analysis**
```
"In my NEET mock, I got wrong: [list topics].
 For each wrong topic:
 - Is it a NCERT knowledge gap?
 - Is it a conceptual misunderstanding?
 - Is it an assertion-reason format issue?
 Create a targeted revision plan."
```

**M-02 — Speed Analysis**
```
"I spent too much time on Biology in my mock.
 Typical time per NEET Biology question should be 45–60 seconds.
 What strategies help read-and-answer Biology MCQs faster?
 How do I identify the answer type quickly?"
```
