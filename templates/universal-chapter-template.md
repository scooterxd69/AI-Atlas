# ⚡ SECTION 9 — PROBLEM SOLVING FRAMEWORK

---

## 🔧 9.1 — Universal Problem-Solving Process

```
STEP 1: READ & IDENTIFY
  └─ Read the problem twice (slowly the second time)
  └─ Identify: What is GIVEN? What is ASKED?
  └─ Identify: Which concept / formula does this test?

STEP 2: DRAW & ORGANIZE
  └─ Draw a diagram (always, even for simple problems)
  └─ List all given values with units
  └─ Mark what needs to be found

STEP 3: PLAN THE APPROACH
  └─ Which formula / law / principle applies?
  └─ Is there a direct path or do you need intermediate steps?
  └─ Estimate the order of magnitude of the answer

STEP 4: SOLVE STEP-BY-STEP
  └─ Write every step — never skip in practice
  └─ Keep track of units throughout
  └─ Use algebra before substituting numbers

STEP 5: VERIFY THE ANSWER
  └─ Check units in the final answer
  └─ Does the magnitude make physical / chemical sense?
  └─ Check with a limiting case if possible
  └─ Re-read the question — did you answer what was asked?
```

---

## 🔧 9.2 — Beginner Approach

> For students new to this chapter.

- Start with direct formula application problems only
- Always write the formula before substituting values
- Work only with given values — do not introduce unknowns without reason
- Solve 5 easy problems per concept before moving forward
- Time limit: no rush — understand each step completely

**Beginner Problem Checklist:**
- [ ] I identified the correct formula
- [ ] I checked all given values and their units
- [ ] I substituted correctly
- [ ] I simplified step by step
- [ ] My final answer has correct units
- [ ] The number makes sense

---

> For students aiming for JEE Main / NEET level.

- Combine multiple concepts in one problem
- Practice recognizing which formula to use from the question structure alone (keyword recognition)
- Aim for under 3 minutes per JEE Main question
- Learn standard "tricks" and shortcuts for this chapter (listed below)

**Common Shortcuts for This Chapter:**

|---|---|---|

## 🔧 9.4 — Advanced Approach

> For students aiming for JEE Advanced / top NEET scores.

- Multi-step problems requiring 3+ concepts simultaneously
- Work with parametric / symbolic cases before plugging numbers
- Develop physical/chemical intuition to predict answer direction before solving
- Practice "reverse engineering" — given an answer, trace back to find missing values
- Time limit: 5–8 minutes per JEE Advanced problem

**Advanced Problem Strategies:**
1. **Boundary / Limiting Cases:** Check the formula at extremes (e.g., t→0, θ→90°) to verify it behaves as expected
2. **Symmetry Exploitation:** Use symmetry to reduce unknowns
3. **Dimensional Analysis:** If you forget a formula, reconstruct it from units
4. **Energy Methods:** Often faster than force / kinematics approaches
5. **Parameterization:** Introduce a parameter to generalize, then optimize

## 🔧 9.5 — Error Detection Checklist

> Run this checklist on every problem where you got the wrong answer.

```
ERROR DETECTION CHECKLIST

□ 1. Did I misread the question? (Read again, slowly)
□ 2. Did I confuse similar-looking symbols? (e.g., v vs ν, b vs β)
□ 3. Did I use the wrong formula for this situation?
□ 4. Did I fail to convert units before substituting?
□ 5. Did I drop a negative sign somewhere?
□ 6. Did I make an arithmetic error? (Recalculate from scratch)
□ 7. Did I apply a formula outside its valid conditions?
□ 8. Did I confuse two similar concepts? (e.g., speed vs velocity)
□ 9. Did I miss a given value hidden in the question text?
□ 10. Did I answer the right question? (Check what was asked)

Fix for next time: ____________________________________
```

# ⚡ SECTION 12 — REVISION SYSTEM

## 🔄 12.1 — Revision Protocol Overview

```
REVISION CYCLE CHART

First Study (Day 0)
│
├─── 1-Day Revision (Day 1)      ← Consolidate memory before it fades
│
├─── 7-Day Revision (Day 7)      ← First spaced recall
│
├─── 30-Day Revision (Day 30)    ← Medium-term consolidation
│
└─── 90-Day Revision (Day 90)    ← Long-term mastery check (pre-exam)
```

---

## 🔄 12.2 — 1-Day Revision (24 Hours After First Study)

**Time Required:** 20–30 minutes
**Goal:** Lock in initial learning. Identify gaps.

**Steps:**
1. [ ] Review your one-page notes (5 min)
2. [ ] Cover your notes and recall all concepts on blank paper (10 min)
3. [ ] Check what you missed — highlight gaps in red
4. [ ] Re-read only the sections you forgot (5 min)
5. [ ] Solve 3 easy practice problems from memory (10 min)
6. [ ] Update your active recall Q&A sheet

**Quick Check Questions:**
- Can I state the core definition of every concept?
- Can I write all formulas from memory?
- Can I solve one basic problem without looking at notes?

---

## 🔄 12.3 — 7-Day Revision (One Week Later)

**Time Required:** 45–60 minutes
**Goal:** First true spaced repetition. Strengthen weak areas.

**Steps:**
1. [ ] Attempt your active recall Q&A sheet from memory (15 min)
2. [ ] Score yourself and note which questions failed
3. [ ] Re-study failed concepts only (10 min)
4. [ ] Solve 5 mixed-difficulty problems (20 min)
5. [ ] Solve 3 PYQs from this chapter (15 min)
6. [ ] Review your mistake log and add any new errors

**Progress Check:**
- [ ] I can recall all formulas without looking
- [ ] I solved PYQs correctly on first attempt
- [ ] I identified my 2–3 weakest sub-topics

---

## 🔄 12.4 — 30-Day Revision (One Month Later)

**Time Required:** 60–90 minutes
**Goal:** Consolidate knowledge. Reach exam-ready level.

**Steps:**
1. [ ] Do a timed mini-test: 10 MCQs from this chapter (15 min, closed book)
2. [ ] Review wrong answers and trace the error type
3. [ ] Solve 5 JEE Main level problems under exam conditions
4. [ ] Attempt 2 JEE Advanced / hard NEET problems
5. [ ] Review your mistake log — have repeat mistakes been fixed?
6. [ ] Update your one-page notes with any new insights
7. [ ] Rebuild formula sheet from memory — cross-check

**30-Day Target:** Achieve ≥80% accuracy on a 10-question test

---

## 🔄 12.5 — 90-Day Revision (Three Months Later — Pre-Exam)

**Time Required:** 90–120 minutes
**Goal:** Exam simulation. Maximum confidence.

**Steps:**
1. [ ] Solve a mini mock test: 15 questions in 20 minutes (strict timing)
2. [ ] Full mistake log review — classify all errors as "fixed" or "recurring"
3. [ ] Re-solve your 3 hardest-ever problems from this chapter
4. [ ] Teach the chapter (explain top 3 concepts out loud or in writing)
5. [ ] Review peak-anxiety points — the tricky bits that always confuse you
6. [ ] Final formula sheet review — ensure 100% recall

**90-Day Target:** Achieve ≥90% accuracy on a 15-question timed test
