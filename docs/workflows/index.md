## 🚀 Most Popular Workflows

### 🔥 New Chapter Workflow

**Use when:** You're starting a new chapter from scratch

**Steps:**

**Step 1: Get the Overview (5 min)**
```
I'm starting [CHAPTER NAME] in [SUBJECT] for [EXAM].
Give me a 5-minute overview:
- What is this chapter about?
- Why does it matter for my exam?
- What do I need to know before starting?
- What are the 3 most tested concepts?
```

**Step 2: Build the Structure (10 min)**
```
Create a complete topic tree for [CHAPTER] that shows:
- Main sections
- Sub-topics under each
- Estimated difficulty (Easy/Medium/Hard)
- JEE/NEET relevance (High/Medium/Low)
```

**Step 3: Study Each Section** (use the study loop)
```
Explain [SECTION] from [CHAPTER]:
1. Core concept in simple terms
2. Key formula/rule
3. One worked example
4. Common traps in exams
5. 2 practice questions
```

**Step 4: Connect the Dots (15 min)**
```
Now I've studied [CHAPTER]. Help me:
1. Summarize the 5 most important points
2. Show how concepts within the chapter connect
3. Link this to chapters I've studied before
4. Predict the 3 most likely exam questions
```

**Step 5: Test Yourself (20 min)**
```
Quiz me on [CHAPTER] with 10 questions:
- 3 easy (recall)
- 4 medium (application)
- 3 hard (JEE/NEET level)
After I answer each, tell me if I'm right and why.
```

---

## 💡 Workflow Tips

- Don't skip steps — the sequence matters
- Use **Claude** for explanation-heavy steps
- Use **Perplexity** for fact-checking or finding resources
- Always do Step 5 (test yourself) — don't stop after reading
- Save good AI conversations for later review
