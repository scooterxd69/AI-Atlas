## For Class 9–10 Students

**S-01 — Concept Explainer (Simple)**
```
"I'm a Class [9/10] student. Explain [TOPIC] simply.
 Use a real-life example I can relate to.
 Then tell me what questions are asked about this in board exams."
```

**S-02 — Board Answer Writer**
```
"The board exam question is: [QUESTION] ([X] marks)
 I'm in Class [10/12]. Write the model answer.
 Highlight keywords in bold. Show diagram placement if needed."
```

**S-03 — Formula Sheet**
```
"Create a formula sheet for [CHAPTER] for Class [9/10].
 Only formulas needed for board exams.
 Include: formula, what each letter means, units."
```

**S-04 — Science Diagram Guide**
```
"List all diagrams CBSE has asked in the last 5 years
 for Class [10] [subject] from [chapter].
 Tell me what labels to draw and how many marks each carries."
```

**S-05 — Difference Tables**
```
"Create a difference table between [CONCEPT A] and [CONCEPT B]
 for Class [10] board exams.
 Include at least 5 differences.
 Use the CBSE tabular format."
```

**S-06 — Numerical Practice**
```
"Give me 5 numerical problems from [CHAPTER]
 at CBSE Class [10] board level.
 Show the solution step by step.
 Include the formula first."
```

**S-07 — Long Answer Guide**
```
"The 5-mark board question on [TOPIC] asks me to explain it.
 Structure the perfect answer with:
 - Introduction (1 line)
 - Main content (with sub-points)
 - Diagram reference if needed
 - Conclusion (1 line)"
```

**S-08 — Chapter Summary**
```
"Summarise NCERT Chapter [name] for Class [9/10] in
 10 bullet points. Focus on what's likely to be asked
 in the board exam. Simple language."
```

**S-09 — MCQ Practice**
```
"Create 10 MCQs from [CHAPTER] for Class [10]
 at CBSE board level. Mix concept-based and
 definition-based questions. Include answer key."
```

**S-10 — Common Mistakes**
```
"What are the 5 most common mistakes Class [10]
 students make when answering questions about [TOPIC]
 in CBSE board exams? How do I avoid them?"
```

## For Class 11–12 Students

**S-11 — Derivation Guide**
```
"This derivation is important for Class 12 [Subject] boards:
 [derivation name]. Break it down step by step.
 Number each step. Explain WHY each step is valid.
 What diagram is needed, if any?"
```

**S-12 — Important Questions List**
```
"List the 10 most important questions from [Chapter]
 for CBSE Class 12 [Subject] board exam.
 Include: question, expected marks, key points to include."
```

**S-13 — Practical Viva Preparation**
```
"I have my Class 12 [Subject] practical exam.
 For experiment: [experiment name]
 Give me 10 expected viva questions with answers."
```

**S-14 — Case Study Practice**
```
"CBSE now includes case-study questions.
 Create a case study question on [TOPIC] for Class [11/12].
 Include: the passage, 4 sub-questions, answer key."
```

**S-15 — Chapter Revision (30 min)**
```
"Give me a 30-minute revision plan for [CHAPTER]
 for Class [12] board exam. What to read, what to practise,
 what to memorise. Time it so I finish in 30 minutes."
```
