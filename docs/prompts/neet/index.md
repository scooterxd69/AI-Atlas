## 🌱 Biology Prompts → [Full List →](#/prompts/neet/biology-prompts)

**Human Physiology (High Weightage):**
```
I'm studying [SYSTEM] (e.g., Digestive/Respiratory/Circulatory) for NEET.
From NCERT, help me:
1. List every organ and its function
2. Step-by-step process (e.g., digestion from mouth to absorption)
3. Key enzymes, hormones, or chemicals involved
4. Important numerical values to memorize (pH, volume, temperature)
5. 10 NEET-style MCQs from this system
```

**Plant Kingdom / Animal Kingdom:**
```
Create a comparison table for NEET covering [TOPIC]:
- Key differences between [GROUP A] and [GROUP B]
- Examples of each
- Special characteristics to remember
- Previous year NEET questions on this comparison
Format it for quick revision — I have 30 minutes.
```

**Genetics & Evolution:**
```
Explain [CONCEPT from Genetics] for NEET:
1. Core concept with simple definition
2. Mendel's laws connection (if relevant)
3. How to solve problems (with formula if applicable)
4. Exceptions and special cases NEET tests
5. 5 MCQs with explanations
```

## 💡 NEET-Specific Tips

> NEET is 90% NCERT. Your AI prompts should always reference NCERT.

Use this at the end of any biology prompt:
```
...and specifically tell me: is this from NCERT? Which chapter and line?
I don't want to study beyond NCERT for Biology.
```
