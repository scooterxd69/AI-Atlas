The Universal Doubt Solver

Subject: [SUBJECT]
Exam: [JEE/NEET/CBSE Class X]
Concept I'm stuck on: [DESCRIBE EXACTLY]

My current (wrong) understanding: [WHAT YOU THINK IT MEANS]
What confused me: [WHERE YOU GOT LOST]

Please:
1. Identify my exact misconception
2. Re-explain from scratch using a different angle
3. Use an analogy from everyday life
4. Give me a quick test to check I've understood

The 5-Why Explainer

Explain [CONCEPT] to me.
If I say "but why?" to any explanation, keep going deeper.
I want to understand WHY, not just WHAT.
Start simple and build up. I'll ask "why?" when I want more.

🔄 Revision → Full Guide →

Pre-Exam Rapid Revision

I have [X hours] before my [SUBJECT] exam.
Topics I need to cover: [LIST]
Difficulty: [which ones are weak/strong]

Create:
1. Priority order (most marks per minute)
2. For each topic: 5 key points only
3. Memory tricks for anything formula-heavy
4. 3 predicted questions per topic

Spaced Repetition Session

I studied [TOPIC] [X days] ago. Quiz me with 10 questions to check retention:
- 4 easy recall questions
- 4 application questions  
- 2 challenging questions
After I answer each, give instant feedback.
At the end, tell me what I've forgotten and need to re-study.

📝 Note Making → Full Guide →

CORNELL Notes Generator

Convert this content into Cornell Notes format for [SUBJECT]:

[PASTE YOUR CONTENT]

Format:
- Cue Column: key questions/terms on the left
- Notes Column: main points on the right
- Summary: 3-sentence summary at bottom
- 3 probable exam questions

Mind Map Prompt

Create a text-based mind map for [TOPIC]:
Central concept → main branches → sub-branches → key facts
Show connections between branches where they exist.
Max depth: 3 levels.

✍️ Essay & Answer Writing → Full Guide →

Essay Structure Builder

I need to write a [TYPE] essay/answer on: [TOPIC]
Exam: [EXAM NAME]
Word/marks limit: [LIMIT]

Give me:
1. Optimal structure for this type of question
2. Key points to include (that evaluators look for)
3. Opening line options (3 choices)
4. What NOT to write (common mistakes)
5. A sample paragraph to show the style
