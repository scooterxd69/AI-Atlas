How to Use These Prompts

  1. Open any AI tool — Claude (recommended), ChatGPT, or Gemini
  2. Copy any prompt — Use the copy button on any prompt block
  3. Fill the brackets — Replace [TOPIC], [CHAPTER] etc. with your content
  4. Iterate — Ask follow-up questions based on the AI's response
  5. Save what works — Keep effective conversations for revision
Best tool: Claude (free tier) gives the most detailed, structured educational responses. See Claude for Students guide.
