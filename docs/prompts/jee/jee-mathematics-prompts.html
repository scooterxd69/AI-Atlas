Calculus Prompts

P-MATH-01 — Integration Technique Identifier

"I have this integral: [integral].
 Don't solve it yet. First identify:
 1. Which technique should be used and why
 2. What clues in the integral indicate this technique
 3. The step-by-step process
 Then solve it completely."

P-MATH-02 — Definite Integration Properties

"Teach me all the properties of definite integration
 used in JEE. For each property:
 - State the property
 - Prove it briefly
 - Show a JEE question where this property simplifies the problem
 - List when to spot and use it"

Algebra Prompts

P-MATH-03 — Complex Numbers

"Explain the geometric interpretation of complex number operations.
 Multiplication as rotation+scaling, division, conjugate, modulus.
 Then show 3 JEE problems that become trivial with geometric thinking."

P-MATH-04 — P&C Problem Solving

"I have this P&C problem: [problem].
 Before solving: identify the counting principle (Fundamental theorem /
 Permutation / Combination / Inclusion-Exclusion).
 Show the logical structure, then compute."

Coordinate Geometry Prompts

P-MATH-05 — Conic Sections

"Compare all four conic sections (circle, parabola, ellipse, hyperbola).
 For each: standard form, eccentricity, focus/directrix,
 key properties, chord of contact, normal equations.
 Format as a comparison table."
