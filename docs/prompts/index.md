## ⭐ Most-Used Prompts

### 1. The Doubt Solver
Works for any subject, any concept:
```
I'm a [CLASS/EXAM] student struggling with: [CONCEPT/PROBLEM]

My confusion is specifically: [DESCRIBE EXACTLY WHAT YOU DON'T GET]

Please:
1. Diagnose what I'm misunderstanding
2. Re-explain from the correct angle
3. Give me a memorable analogy or example
4. Check my understanding with 2 follow-up questions
```

### 2. The Quick Revision Prompt
Best for the night before a test:
```
I have [X] hours before my [SUBJECT] exam.
My weak topics are: [LIST]
My strong topics are: [LIST]

Create a prioritized revision plan:
- What to revise in what order
- How long to spend on each
- Key formulas/facts to memorize for each topic
- One practice question per topic
```

### 3. The Note Maker
Turn any text into clean notes:
```
Convert this [textbook/lecture] content into study notes for [EXAM]:

[PASTE CONTENT HERE]

Format as:
- Key concepts (with 1-line explanations)
- Formulas and rules (highlighted)
- Important exceptions/special cases
- 5 likely exam questions from this content
```

### 4. The Error Analyzer
When you get a question wrong:
```
I got this question wrong in my [EXAM] mock test:

Question: [PASTE QUESTION]
My answer: [WHAT YOU CHOSE/WROTE]
Correct answer: [WHAT IT SHOULD BE]

Please:
1. Explain exactly why my answer is wrong
2. Walk through the correct solution
3. Identify the conceptual gap I need to fix
4. Give me 3 similar questions to practice
```

### 5. The Chapter Explainer
For new chapters:
```
I'm starting Chapter [NAME] in [SUBJECT] for [EXAM].
I know: [WHAT YOU ALREADY KNOW RELATED TO THIS]
I don't know: [STARTING FROM SCRATCH or specific gaps]

Please give me:
1. A 5-minute overview of the entire chapter
2. The 3 most important concepts I must master
3. How this chapter connects to other chapters
4. The most common exam questions from this chapter
5. What to focus on vs what to skim
```

## 💡 Pro Tips for Using Prompts

1. **Be specific** — Replace every `[BRACKET]` with real details
2. **Use Claude** for deep explanations, ChatGPT for quick answers
3. **Follow up** — "Explain point 2 in more detail"
4. **Save good conversations** — bookmark/export them
5. **Iterate** — If you don't like the output, say "Try again, but simpler"
