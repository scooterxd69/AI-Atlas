## 💡 Why AI for School?

School students often miss out on personalized learning. AI changes that:

- **Instant doubt clearing** — no waiting for the teacher
- **Concept re-explanation** — in different ways until it clicks
- **Custom practice questions** — based on your weak areas
- **Essay and assignment help** — with feedback and improvements
- **Exam pattern analysis** — predict what's likely to come

---

## 🧠 Top School AI Prompts

**Concept Doubt Prompt:**
```
I'm a Class [X] CBSE student. I don't understand [CONCEPT] from [SUBJECT].
Please explain it:
1. In the simplest possible language
2. With a real-life example or analogy
3. With a diagram (text-based) if helpful
4. Follow up with 3 questions to check if I've understood
```

**Assignment Help Prompt:**
```
My CBSE Class [X] assignment topic is: [TOPIC]
Subject: [SUBJECT]
Help me:
1. Outline the key points I should cover
2. Suggest a structure for my answer
3. Highlight what the examiner expects
4. Give me 3 important points to include that most students miss
Note: I want to understand and write it myself, just guide me.
```
