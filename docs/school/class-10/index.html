🧠 Class 10 Top Prompts

Science Concept:

I'm a Class 10 CBSE student studying [TOPIC] in Science.
Explain it for board exams:
1. Core concept (simple language)
2. Important definitions (mark-worthy)
3. Draw/describe the key diagram
4. Common board exam question types
5. 5-mark answer structure for "Explain [TOPIC]"

Maths Problem:

I'm stuck on this Class 10 Maths problem: [PASTE PROBLEM]
Chapter: [CHAPTER NAME]
I tried: [YOUR ATTEMPT]

Show me:
1. Which concept/formula applies
2. Step-by-step solution
3. How to avoid this mistake in the exam
4. 2 similar practice problems

Social Science Answer:

CBSE Class 10 Social Science exam question: [QUESTION]
Marks: [X marks]
Subject: [History/Geography/Civics/Economics]

Write a model answer that:
- Exactly fits the mark scheme
- Uses proper keywords
- Has the right structure (intro, points, conclusion)
- Would score full marks

✅ Board Exam Strategy with AI

Week before exams:

My Class 10 [SUBJECT] board exam is in [X] days.
Important chapters: [list]
Weak chapters: [list]

Build a day-by-day revision plan:
- What to revise each day
- How long per chapter
- What to do the night before
- Morning of exam tips
