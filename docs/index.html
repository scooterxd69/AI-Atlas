Sample AI Prompts

Copy these directly into ChatGPT, Claude, or any AI. 125+ more in the full library.

Physics · JEE
"Explain the concept of [topic] as if I'm a JEE aspirant who just saw it for the first time. Start with the physical intuition, then the formula, then show me a PYQ-style question with full solution."
→ Concept clarity + application
Biology · NEET
"Create a high-yield summary table for [NCERT chapter] with the most tested facts, common confusions, and 5 predicted MCQs with explanations."
→ Revision + MCQ practice
Maths · JEE
"I attempted this problem and got stuck at step 3: [problem]. Walk me through the correct approach step-by-step, explain where my thinking went wrong, and give me a similar problem to practice."
→ Error analysis + follow-up
Chemistry · JEE/NEET
"Generate a mnemonics sheet for [topic] — reactions, exceptions, and any hard-to-remember sequences. Make it visual and memorable."
→ Memory aids + retention
Study Planning
"I have [X weeks] before my exam. My strong topics are [list], weak topics are [list]. Create a day-wise revision plan that maximises score improvement."
→ Personalised scheduling
CBSE Boards
"Convert this NCERT paragraph into a 5-point answer format suitable for a 5-mark board exam question. Keep the language clear and examinable."
→ Answer writing practice
