## 🧠 Top NEET AI Prompts

**Biology Diagram Prompt:**
```
I'm studying [TOPIC] for NEET. Create a detailed text-based diagram/flowchart showing:
1. All major components and their roles
2. Key processes in sequential order
3. Important NEET facts to remember for each part
4. Common question patterns from this topic
Format it clearly so I can use it for quick revision.
```

**NCERT Deep-Dive Prompt:**
```
I'm reading NCERT Chapter [X] on [TOPIC] for NEET.
Help me extract:
1. Every fact that could be a NEET question
2. Key terms and definitions
3. Important diagrams/processes to memorize
4. Typical NEET question patterns from this chapter
5. Common errors students make
```
