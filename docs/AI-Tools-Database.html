Tools marked 🆓 are completely free. Tools marked 🔁 have a free tier with paid upgrades.

Recommended Tool Stacks by Student Type

JEE Aspirant (Class 11–12)

Core daily stack: Claude + Wolfram Alpha + Desmos + Anki + Notion For Physics: PhET + Claude + Wolfram Alpha For Chemistry: Claude (Organic mechanisms) + PubChem + Anki For Maths: Desmos + GeoGebra + Symbolab + Claude

NEET Aspirant (Class 11–12)

Core daily stack: Claude + Anki + Notion + NCERT official For Biology: Visible Body + BioRender + Khan Academy + Claude For Physics/Chem: PhET + Wolfram Alpha + Claude

Class 9–10 School Student

Core daily stack: Khan Academy + Claude + Photomath + Canva For exam prep: Quizlet + Google Calendar + Grammarly For projects: Canva + Coggle + Piktochart

Beginner to AI (any student)

Start with these 5: Claude + Khan Academy + Anki + Desmos + Notion All five are free, beginner-friendly, and immediately useful for any student.
