## How to Upload Your NCERT / Textbook Pages

1. Take a photo of the page (or use a PDF)
2. Click the paperclip icon in Claude
3. Upload the image/PDF
4. Ask your question about the content

**Example prompt with image upload:**
```
[Upload: photo of NCERT Chemistry page on Mole Concept]

"Explain the derivation on this page in simpler terms.
 Then give me 3 practice problems at increasing difficulty
 with solutions."
```

## 20 Ready-to-Use Claude Prompts

### Learning New Concepts

**Prompt 1 — First Principles Explainer**
```
"Explain [TOPIC] for JEE from absolute first principles.
 Assume I know basic [PREREQUISITE] but nothing else.
 Use analogies I can visualise. Then show me how this
 concept appears in JEE questions."
```

**Prompt 2 — Concept Builder**
```
"I'm studying [CHAPTER] for JEE/NEET. I understand the
 basic definition but I don't understand WHY it works.
 Explain the underlying reasoning, then show me a common
 misconception students have and why it's wrong."
```

**Prompt 3 — Visual Explanation Request**
```
"Explain [TOPIC] as if you're drawing a diagram.
 Describe each element, what it represents, and how
 the parts connect. I'll draw it as you describe."
```

### Doubt Solving

**Prompt 4 — Specific Doubt**
```
"In [CHAPTER], I don't understand why [SPECIFIC THING].
 I thought [MY WRONG REASONING]. Where is my thinking wrong?
 Correct my understanding without just giving me the answer."
```

**Prompt 5 — Derivation Breakdown**
```
"Break down the derivation of [FORMULA/RESULT] for me.
 Do it step by step. At each step, explain WHY that
 mathematical operation is valid and what it means physically."
```

### Revision

**Prompt 6 — Quick Formula Sheet**
```
"Create a complete formula sheet for [CHAPTER] for JEE.
 Include: formula, variables definition, when to use it,
 common mistakes. Format as a clean table."
```

**Prompt 7 — 10-Point Revision Checklist**
```
"Create a 10-point revision checklist for [CHAPTER].
 Each point should be something I can actively verify
 I understand, not just 'read the chapter'."
```

**Prompt 8 — Rapid Fire Questions**
```
"Give me 15 rapid-fire conceptual questions on [CHAPTER].
 After I answer each one (say 'next'), tell me if I'm
 right and correct any misunderstanding briefly."
```

### PYQ Analysis

**Prompt 9 — PYQ Pattern Analysis**
```
"Based on JEE Main PYQs, what are the 5 most commonly
 tested concepts from [CHAPTER]? For each concept, give me
 one example question type and what the examiner is testing."
```

**Prompt 10 — Difficulty Level Training**
```
"Create 3 questions on [TOPIC]:
 1. JEE Main level (straightforward application)
 2. JEE Advanced level (multi-concept)
 3. JEE Advanced level (tricky/non-standard)
 Solve all three with full working."
```

### Mock Analysis

**Prompt 11 — Mock Error Analysis**
```
"I got [TOPIC/QUESTION TYPE] wrong in my mock test.
 My approach was: [YOUR APPROACH]
 The correct answer was: [CORRECT ANSWER]
 Diagnose exactly where my reasoning failed and
 what I need to revise to avoid this mistake."
```

**Prompt 12 — Weakness Identification**
```
"I consistently struggle with [TOPIC AREA].
 Ask me 5 diagnostic questions to identify exactly
 which sub-concept I'm weak in. Then create a
 targeted 2-hour study plan to fix it."
```

## Advanced Tips

**Upload your mock answer sheet:**
```
[Upload: photo of your mock test answers]
"Analyse my answers. Identify my error patterns.
 What topics do I need to focus on most?"
```

**Use Claude as a Socratic tutor:**
```
"Don't give me the answer directly. Ask me guiding
 questions that help me figure out [CONCEPT] myself."
```

**Create personalised notes:**
```
"Based on our conversation, create structured notes
 on [TOPIC] that I can save and revise. Include:
 key concepts, formulas, common pitfalls, and
 3 example questions."
```
