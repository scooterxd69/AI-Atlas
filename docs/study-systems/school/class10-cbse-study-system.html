3-Month Revision Plan

Month 1: Complete syllabus coverage

  • 2 Science chapters per week
  • 1 Maths chapter per week (with all exercise problems)
  • Social Science: one unit per week

Month 2: Weak topic recovery + practice

  • Identify weak chapters via mock tests
  • Use WF-08 Weak Topic Recovery
  • Start solving CBSE sample papers

Month 3: Board exam sprint

  • Daily CBSE sample paper (2018–2024)
  • WF-13 Last 7 Days Sprint in final week
  • Focus on diagram practice (Science) and proof writing (Maths)

Board Answer Writing with AI

"I'm appearing for CBSE Class 10 [subject].
 The question is: [QUESTION]
 It is worth [X] marks.
 Write a model answer following the CBSE marking scheme.
 Highlight the keywords the examiner is looking for."

Diagram Practice Prompt

"List all diagrams that CBSE Class 10 Science board exam
 has asked in the last 5 years. For each diagram:
 topic, what it shows, what labels are required,
 how many marks it typically carries."
