## 💡 How to Use a Study System

1. **Pick the system** that matches your current exam/goal
2. **Read the full system once** before starting
3. **Set up your AI tool** (Claude recommended for depth)
4. **Follow Phase 1** — don't skip ahead
5. **Track your progress** using the system's checkpoints
6. **Adapt as needed** — the system is a framework, not a prison
