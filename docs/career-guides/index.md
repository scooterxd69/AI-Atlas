## 💡 Use AI for Career Planning

You can use AI to research careers before committing:

```
I scored [RANK] in JEE Main [YEAR].
I'm interested in [FIELD/BRANCH PREFERENCES].
I prefer [CITY PREFERENCE or "anywhere"].

Help me:
1. List colleges I can realistically target
2. Rank branches by placement, research, and future scope
3. Tell me what to expect in each option
4. Ask me 3 questions to help narrow down further
```
