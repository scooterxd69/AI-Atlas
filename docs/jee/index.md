## 🧠 Top JEE AI Prompts

Copy these directly into Claude, ChatGPT, or Gemini:

**Concept Explanation Prompt:**
```
Explain [CONCEPT] as if I'm a JEE Main aspirant. Include:
1. Core principle in simple language
2. Key formula and when to use it
3. Common mistakes in JEE questions
4. One solved example with steps
5. Three practice questions (with difficulty: Easy/Medium/Hard)
```

**Error Analysis Prompt:**
```
I got this JEE question wrong: [paste question]
My approach was: [explain what you tried]
Please:
1. Identify exactly where my thinking went wrong
2. Explain the correct approach step by step
3. Tell me what concept I need to strengthen
4. Give me 2 similar questions to practice
```
