## 🎯 Which Tool for Which Task?

| Task | Best Tool | Why |
|---|---|---|
| Explain a complex concept | Claude | Deepest reasoning |
| Quick doubt clarification | ChatGPT | Fast, accurate |
| Research a topic with sources | Perplexity | Cites sources |
| Study from your own PDF/notes | NotebookLM | Trained on your docs |
| Solve a math problem | ChatGPT / Claude | Both excellent |
| Write/improve an essay | Claude | Best writing quality |
| Create flashcards | ChatGPT | Easy to format |
| Understand a diagram | Gemini | Strong vision model |
| Hindi-medium students | ChatGPT / Gemini | Better Hindi support |
