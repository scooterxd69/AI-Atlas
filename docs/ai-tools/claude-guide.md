## 💡 How to Get the Best Results

### Tip 1: Tell Claude your context
```
I'm a Class 12 student preparing for JEE Main. I have a strong 
understanding of Class 11 physics but Class 12 topics are new to me.
Explain at that level.
```

### Tip 2: Ask for what you need specifically
❌ "Explain Newton's Laws"  
✅ "Explain Newton's Third Law specifically for problems involving two blocks 
connected by a string — this is where I get confused in JEE problems"

### Tip 3: Push for clarity
If an explanation isn't clear:
- "Explain point 2 again, but simpler"
- "I still don't understand [X]. Try a different analogy"
- "Show me a worked example for this"

### Tip 4: Use it as a conversation
Don't start fresh each session for the same topic. Build on previous messages:
- "Now apply that concept to this problem..."
- "What's the exception to the rule you just explained?"
- "Connect this to what you told me about [topic] earlier"

## ⚠️ When NOT to Use Claude

- **Real-time search** — Claude's knowledge has a cutoff date. Use Perplexity for current info.
- **Solving photos of questions** — Use ChatGPT or Gemini (better image processing)
- **Hindi-medium content** — Claude works in Hindi but ChatGPT/Gemini are better
- **When you need source citations** — Use Perplexity

## 🔥 Power User Techniques

### Create a Study Persona
Paste this at the start of a study session:
```
For this conversation, you are my personal JEE tutor. You know my level: 
[describe your level]. Your teaching style should be: [Socratic/direct/
example-heavy]. When I make mistakes, point them out directly. 
Never give me the answer without making me think first.
```

### The Feynman Test
```
I'm going to explain [CONCEPT] to you as if you know nothing.
After I finish, tell me: 
1. What did I explain correctly?
2. Where was I wrong or incomplete?
3. What important points did I miss?
Ready? Here goes: [YOUR EXPLANATION]
```
