# Content Standards

Every asset must:

- Be beginner friendly
- Be practical
- Include examples
- Use markdown
- Be actionable

Avoid:
- Hype
- Buzzwords
- Marketing language

Study Systems Must Include:
- Learning
- Notes
- Revision
- Question Solving
- Mock Analysis
- Improvement Loop

Prompts Must Include:
- Purpose
- Prompt
- Example
- Tips
